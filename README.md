## 📖 Descripción del Curso

Este repositorio contiene todo el material práctico, guías interactivas y libretas de Google Colab desarrolladas para la asignatura de **Programación de Computadores / Fundamentos de Programación**.

El curso está diseñado con un enfoque 100% aplicado a la ingeniería, llevando a los estudiantes desde los fundamentos del pensamiento algorítmico hasta el despliegue de aplicaciones web interactivas y la resolución de problemas específicos de sus áreas (estática de vigas y termodinámica de secadores solares). 

Nuestro objetivo no es solo enseñar a escribir código, sino formar ingenieros capaces de construir sus propias herramientas tecnológicas.

---

## 🚀 Estructura del Semestre

El contenido está dividido en 16 semanas, estructuradas en cuatro grandes módulos:

### 🧩 Módulo 1: Fundamentos de Lógica Computacional
* **Semana 1:** Introducción a la Programación y al Entorno de Trabajo (Modelo E-P-S).
* **Semana 2:** Variables, Tipos de Datos y Operaciones Matemáticas.
* **Semana 3:** Estructuras Condicionales (Toma de decisiones lógicas).
* **Semana 4:** Bucles de Control (Automatización de tareas iterativas).
* **Semana 5:** Funciones en Python (Modularidad y reutilización de código).

### 🌐 Módulo 2: Interfaces Web y Herramientas Profesionales
* **Semana 6:** De la Lógica a la Web (Introducción a Streamlit).
* **Semana 7:** Entorno Profesional Local (VSCode, Git y GitHub).
* **Semana 8:** Despliegue en la Nube (Streamlit Community Cloud).

### 📊 Módulo 3: Análisis Numérico y Ciencia de Datos
* **Semana 9:** Análisis Numérico y Visualización Gráfica (`NumPy` y `Matplotlib`).
* **Semana 10:** Manipulación de Bases de Datos con `Pandas`.
* **Semana 11:** Visualización Estadística Interactiva (Construcción de Dashboards).

### ⚙️ Módulo 4: Especialización y Proyecto Final Integrador
En esta etapa, el curso se bifurca para atender las necesidades específicas de cada programa:
* **Ruta A (Ingeniería Civil):** Álgebra simbólica, mecánica del medio continuo, cálculo de reacciones y diagramas de cortante/momento flector utilizando `SymPy`.
* **Ruta B (Ing. Agrícola y Agroindustrial):** Algoritmos de contenido de humedad, balances de masa, termodinámica y cálculo de caudales de aire para secadores solares.
* **Semanas 14 a 16:** Arquitectura de software, Taller de Integración Gráfica (Backend + Frontend) y Sustentación Final de las aplicaciones web.

## 💡 Cómo usar este repositorio

Los archivos con extensión `.ipynb` son libretas interactivas (Notebooks). Existen varias formas de utilizarlas:

1. **En la Nube (Recomendado):** Haz clic en cualquier archivo `.ipynb` y utiliza el botón *"Open in Colab"* para ejecutar el código directamente en tu navegador sin instalar nada.
2. **Entorno Local:** Clona este repositorio en tu computador utilizando Git (`git clone https://github.com/justorfc/Programacion_2026_1.git`) y abre la carpeta en Visual Studio Code. Asegúrate de tener instalada la extensión de Jupyter y las librerías mencionadas en el archivo `requirements.txt`.
3. **Códigos Fuente:** Los archivos `.py` corresponden a los scripts base para el despliegue de las aplicaciones web interactivas en Streamlit.
